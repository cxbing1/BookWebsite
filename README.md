 图书网站
 ===
 
一、环境<br>
服务器：Apache2.4<br>
后台语言：PHP7.2<br>
数据库：Mysql5.6.26<br>


二、功能设计<br>
1、该图书网站实现以下功能：<br>
（1）登录/注册功能<br>
（2）用户信息显示功能<br>
（3）书籍信息显示功能<br>
（4）书籍排行榜功能<br>
（5）书籍分类功能<br>
（6）书籍搜索功能<br>
（7）借书/还书功能<br>
